# 学校の片づけ術 チェックテスト(Web版)

学校片づけアドバイザー・伊藤寛子さんの講演資料「片づけから始める学校の働き方改革」
(群馬県公立小中学校事務職員研究会向け)をもとに、学校の片付けの考え方を9問で確認する
単発テストです。ご本人の許可を得て、講演参加者向けに公開しています。

## 公開URL

**https://tsj-coder.github.io/katazuke-check/**

合言葉を入力しないと問題は表示されません(講演会でご案内した合言葉を使用)。
検索エンジンにも登録されない設定(`noindex`)にしています。

> 合言葉画面はクライアント側チェックのみで、真の意味でのアクセス制限ではありません。
> 「講演参加者以外が検索等で偶然たどり着かないようにする」ための簡易的な入口です。

## 内容

- 全9問・4択。「片づけの効果と方法」「片づく仕組み作り」「みんなで取り組む学校の片づけ」
  の3分野・9論点(「片づけ」の定義、探す時間の無駄の試算、片づけの手順、
  仕組みで考える発想、在庫管理の仕組み、学校が片づかない理由、片づける効果、
  進め方5ステップ、計画と作業の方向性)を出題
- 各問1問ごとに正誤・解説・出典を表示。分野別の正答数も最後に表示
- 「別の類題でもう一度」で、同じ論点の別バリエーションの問題に切り替わる

## 出典

- 伊藤寛子氏 講演資料「片づけから始める学校の働き方改革」(群馬県公立小中学校事務職員研究会)

設問文はこの講演資料の内容をもとにした独自作成です。

## プライバシー

学習コードの登録や進捗保存はありません。回答はすべて回答者の端末内のみで処理され、
外部サーバーへの送信・収集は一切ありません。

## 構成と更新方法

- `index.html` … 画面本体(合言葉ゲート含む)
- `css/style.css` … スタイル
- `js/quiz-engine.js` … 出題・採点ロジック(純関数)
- `js/katazuke.js` … 画面制御・合言葉チェック(合言葉は `GATE_PASSWORD` 定数)
- `questions/katazuke.json` … 問題データ(9論点×2バリエーション)

問題を直したいときは `questions/katazuke.json` を編集して:

```bash
git add -A && git commit -m "問題文修正" && git push
```

push すると GitHub Pages に1分ほどで自動反映されます。
