# 学校の片づけ術 チェックテスト(Web版)

学校片づけアドバイザー・伊藤寛子さん(元小学校教諭31年、著書『学校の片づけ術』学事出版)が
公開しているメソッドから、学校の片付けの考え方を7問で確認する単発テストです。

## 公開URL

**https://tsj-coder.github.io/katazuke-check/**

## 内容

- 全7問・4択。「整理→収納の順番」「不要品がたまる構造」「机の中の片付け手順」
  「事務用品のゾーニング・色分け」「安全(棚の上に棚はあらず)」「子どもが片付けやすい仕組み」
  「書類の処分ルール」の7論点を出題
- 各問1問ごとに正誤・解説・出典を表示。分野別の正答数も最後に表示
- 「別の類題でもう一度」で、同じ論点の別バリエーションの問題に切り替わる

## 出典

- 東洋経済education×ICT「整理収納のプロが語る、忙しい学校も今すぐ可能な『時短をかなえる片付け術』」(2023/4/2)
- 伊藤寛子『学校の片づけ術 今日からできて、「キレイ」が続く仕組み作り』(学事出版)
- note「まき style」掲載インタビュー

設問文はこれらの公開情報をもとにした独自作成です。

## プライバシー

学習コードの登録や進捗保存はありません。回答はすべて回答者の端末内のみで処理され、
外部サーバーへの送信・収集は一切ありません。

## 構成と更新方法

- `index.html` … 画面本体
- `css/style.css` … スタイル
- `js/quiz-engine.js` … 出題・採点ロジック(純関数)
- `js/katazuke.js` … 画面制御
- `questions/katazuke.json` … 問題データ(7論点×2バリエーション)

問題を直したいときは `questions/katazuke.json` を編集して:

```bash
git add -A && git commit -m "問題文修正" && git push
```

push すると GitHub Pages に1分ほどで自動反映されます。
